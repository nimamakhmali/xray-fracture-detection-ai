# X-Ray Fracture Detection AI

Deep learning pipeline for fracture detection and localization in X-ray images, with support for visual explainability.

## Scope
This repository contains the AI/ML part of the project, including:
- data preprocessing
- model training
- inference
- evaluation
- explainability

## Tech Stack
- Python
- PyTorch
- OpenCV
- pydicom
- Albumentations

## Status
Work in progress.

## Notes
This repository uses only public or properly de-identified data.  
It is intended for research and development purposes only, not for clinical use.
